# IBM Data Analyst Capstone Project

This repository contains the final Capstone Project for the **IBM Data Analyst Professional Certificate**, applying the complete data analysis lifecycle to solve a real-world business problem using data-driven insights.

---

📋 Project Overview
The main objective of this project was to analyze a dataset to extract actionable insights, identify key trends, and present the findings through interactive dashboards and executive reports.

🛠️ Technologies and Tools
* **Programming Language:** Python (Pandas, NumPy, Matplotlib, Seaborn)
* **Databases:** SQL / SQLite for data extraction and querying.
* **Data Visualization & Dashboards:** IBM Cognos Analytics
* **Environment:** Jupyter Notebooks

---

🔄 Project Phases (Data Analysis Methodology)

1. **Data Collection & Extraction:** 
   * Importing data from multiple sources (CSV files, SQL databases, and APIs).
2. **Data Wrangling & Cleaning:**
   * Handling missing values, duplicates, and data type corrections.
   * Normalizing and transforming variables for analysis.
3. **Exploratory Data Analysis (EDA):**
   * Using descriptive statistics and visualizations to uncover patterns, correlations, and anomalies inやすい the data.
4. **Dashboard Creation & Visualization:**
   * Developing interactive dashboards using **IBM Cognos Analytics** to visualize key metrics, filter data dynamically, and summarize key performance indicators (KPIs). (https://us3.ca.analytics.ibm.com/bi/?perspective=dashboard&pathRef=.my_folders%2FMy%2Bdashboard&action=view&mode=dashboard&subView=model0000019f6bc31e93_00000000&nav_filter=true)

   * Synthesizing main findings and providing data-driven recommendations for decision-making.
